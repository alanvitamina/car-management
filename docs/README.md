## design/ 目录

`design/` 下的所有文档均为 **V1/V2 设计阶段产物**，在代码开发前编写。

**实际技术栈与设计文档存在差异：**

| 层 | 设计文档 (V1) | 实际实现 |
|----|-------------|---------|
| 后端 | Java 21 + Spring Boot + MyBatis + MySQL | Express + TypeScript + JSON 文件 |
| 缓存 | Redis 7 | 无（MVP 阶段） |
| OCR | 未涉及 | tesseract.js |

**请以根目录 [CLAUDE.md](../CLAUDE.md) 作为当前实现的最新参考。**

- 理解**业务全貌**和规则 → 从《需求冻结清单》和《业务规则清单》开始
